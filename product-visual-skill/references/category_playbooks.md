# Category Playbooks

Use these playbooks to choose the image sequence. If the category is unclear, use `small_home_gadget`.

## Small Home Gadget

Best for mini vacuums, compact appliances, desk accessories, kitchen gadgets, and cleaning tools.

Default image sequence:

1. `cover_main`
2. `premium_hero`
3. `localized_lifestyle`
4. `in_use_model_or_hand`
5. `feature_infographic`
6. `specification_size`
7. `package_contents`
8. `comparison_why_this`
9. `variant_or_detail_closeup`

Key buyer questions:

- What exactly is it?
- How big is it?
- Where do I use it?
- What problem does it solve?
- What is included?
- Are the claims realistic?

## Fashion / Apparel

Default image sequence:

1. `cover_main`
2. `on_model_front`
3. `on_model_side_back`
4. `fit_size_chart`
5. `fabric_detail`
6. `styling_lifestyle`
7. `color_variants`
8. `care_instructions`

Require confirmed size chart before making exact fit graphics.

## Beauty / Personal Care

Default image sequence:

1. `cover_main`
2. `premium_hero`
3. `texture_or_detail`
4. `how_to_use`
5. `benefit_infographic`
6. `routine_lifestyle`
7. `ingredients_or_specs`
8. `package_contents`

Avoid medical, clinical, whitening, acne-cure, anti-aging, or safety claims unless supplied and legally supportable.

## Home Decor

Default image sequence:

1. `cover_main`
2. `room_lifestyle`
3. `scale_reference`
4. `material_detail`
5. `style_variants`
6. `installation_or_placement`
7. `specification_size`
8. `package_contents`

Use local home context such as HDB, condo, compact apartment, or study corner for Singapore.

## Electronics

Default image sequence:

1. `cover_main`
2. `premium_hero`
3. `desk_or_home_setup`
4. `ports_controls_detail`
5. `feature_infographic`
6. `specification_size`
7. `compatibility`
8. `package_contents`

Do not invent power, voltage, battery life, waterproof rating, certifications, or compatibility.
